Public Site Posture

Security & Privacy

Hubsays pages are built as a static public surface with privacy-first defaults, minimal moving parts, and no intentional ad trackers.

Last reviewed: March 2026

Data Handling

No ad trackers are intentionally embedded in site pages. Contact data is user-submitted via email only.

Operational Model

Public artifacts emphasize deterministic workflows, local-first tooling where practical, and explicit guardrails for automation.

Infrastructure Used

The public site is a static HTML surface. Infrastructure currently referenced across the public setup includes Cloudflare at the edge, GitHub for source control, and a minimal publish model designed to avoid third-party trackers or unnecessary embedded scripts.
